# Linux OS EOL Checker Bash Script

This Bash script identifies the current Linux operating system and provides a simulated End-of-Life (EOL) status check. It specifically highlights the EOL of CentOS 7 and 8, emphasizing the critical need for migration to maintain security and stability, as discussed in the accompanying article.

## Language

`bash`

## How to Run

1. Save the code as `check_os_eol.sh`.
2. Make it executable: `chmod +x check_os_eol.sh`.
3. Run it: `./check_os_eol.sh`
